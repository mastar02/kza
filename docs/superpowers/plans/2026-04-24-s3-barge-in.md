# S3 — Barge-in (interrumpir TTS al detectar voz)

**Effort**: 🟡 3-4 h
**Depende**: S2 (response cache simplifica el re-start del TTS post-cancel)
**Branch sugerido**: `feat/s3-barge-in`

## Objetivo

Mientras KZA habla (TTS activo), si el usuario pronuncia una nueva utterance,
cortar el TTS en <200ms y abrir modo listening inmediatamente. Standard de UX
en Alexa/Google.

## Arquitectura

```
Mic stream ──► multi_room_audio_loop.audio_callback
                │
                ├─ TTS activo?
                │   ├─ NO → flujo normal (wake word detection)
                │   └─ SÍ → barge-in check:
                │        - RMS > threshold Y
                │        - silero-VAD activo (no es eco residual del TTS) Y
                │        - duration > 200ms (estable, no pico espurio)
                │        → cancel_tts() + transition to listening
                │
TTS stream ───► ResponseHandler
                ├─ cancel() método → stop playback ASAP, close stream
                └─ estado expuesto: is_speaking: bool
```

### Modificar: `src/pipeline/response_handler.py`

Exponer estado y método cancel:

```python
class ResponseHandler:
    def __init__(self, ...):
        ...
        self._is_speaking = False
        self._playback_task: asyncio.Task | None = None
        self._current_stream = None  # stream handle si hay uno activo

    @property
    def is_speaking(self) -> bool:
        return self._is_speaking

    async def cancel(self) -> bool:
        """
        Interrumpir el TTS actual. Cierra el playback inmediatamente.
        Devuelve True si había algo activo para cancelar.
        """
        if not self._is_speaking:
            return False
        logger.info("⏹  TTS cancel (barge-in)")
        try:
            if self._current_stream is not None:
                self._current_stream.stop()
                self._current_stream.close()
                self._current_stream = None
            if self._playback_task and not self._playback_task.done():
                self._playback_task.cancel()
        except Exception as e:
            logger.warning(f"cancel() cleanup error: {e}")
        self._is_speaking = False
        return True
```

En los métodos que ejecutan TTS (speak, stream, playback):
- Set `self._is_speaking = True` al empezar.
- Set `self._is_speaking = False` al terminar (en finally).
- Guardar `self._current_stream` para poder cancelarlo.

def audio_callback(indata, frames, time_info, status):
    audio_chunk = indata[:, 0].copy()

    # NEW: Barge-in check. Corre solo si TTS está activo.
    if (
        self.barge_in_enabled
        and self._response_handler is not None
        and self._response_handler.is_speaking
    ):
        rms = float(np.sqrt(np.mean(audio_chunk ** 2)))
        is_voice = (
            rms > self.barge_in_rms_threshold
            and rs.echo_suppressor.is_human_voice(audio_chunk)
        )
        if is_voice:
            rs.barge_in_accum_ms += (frames / self.sample_rate) * 1000
            if rs.barge_in_accum_ms >= self.barge_in_min_duration_ms:
                # Trigger barge-in desde audio thread → schedule async cancel
                asyncio.run_coroutine_threadsafe(
                    self._trigger_barge_in(rs),
                    self._loop,

                rs.barge_in_accum_ms = 0.0
                return  # no procesar esto como wake word
        else:

        return  # skip normal flow mientras TTS habla

    # ... resto del callback como está

## Edge cases

- **Echo del TTS contaminando mic**: el `echo_suppressor` actual ya filtra
  output de TTS. Pero barge-in threshold debe ser mayor al eco residual
  (default 0.03 vs típico eco 0.005-0.015). Tunear con pruebas reales.
- **Arranque del TTS tiene delay**: `is_speaking=True` debe setearse ANTES del
  primer sample (en el await del synthesize). Setear al inicio del método.
- **Cancel de un TTS que ya terminó**: `cancel()` debe ser idempotent.
- **Barge-in durante la palabra del usuario**: si el user habla más lento y hay
  una pausa, NO cerrar por accidente. El `min_duration_ms=200` lo protege.
